# WiFi Password Cracker

⚠️ **For Educational Purposes Only**

This Android application demonstrates WiFi security testing concepts for educational purposes.

## Features
- Scan available WiFi networks
- Attempt connection with common passwords
- Display network security type
- Simple and intuitive UI

## Requirements
- Android 5.0+ (API 21+)
- Location permission (required for WiFi scanning)
- WiFi enabled device

## Build Instructions

### Using GitHub Actions
1. Fork this repository
2. Go to Actions tab
3. Run the Android workflow
4. Download APK from artifacts
